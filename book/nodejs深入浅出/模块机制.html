模块编译 #

在 node 中，每个文件模块都是一个对象，定义如下：

js
function Module(id, parent) {
  this.id = id
  this.exports = {}
  this.parent = parent
  if (parnent && parent.children) {
    parent.children.push(this)
  }
  
  this.filename = null
  this.loaded = false
  this.children = []
}

每一个编译成功的模块都会将其文件路径作为索引缓存在 Module._cache 对象上，以提高二次引入的性能。
