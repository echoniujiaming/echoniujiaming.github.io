nodejs 基础 #

解决下载 npm 包速度慢的问题 #

切换 npm 包镜像源

sh
# 查看当前的包镜像源
npm config get registry

# 切换为淘宝镜像源
npm config set registry=https://registry.npm.taobao.org/

# 检查镜像是否下载成功
npm config get registry

或者 使用 nrm 工具进行 npm 管理

sh
npm i nrm -g

# 查看所有可用的镜像源
nrm ls

# 切换使用镜像源
nrm use taobao
